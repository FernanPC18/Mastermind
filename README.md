# MASTERMIND

## Historias de usuario

- El usuario pone una combinación de 4 colores de entre 6 colores distintos (los cuales pueden repetirse a lo largo del código)
- La máquina intenta adivinarlo enum máximo de 12 intentos
- Por cada solución, se ofrecerá una corrección mostrando información sobre cada color, con 3 posibles estados posibles para cada uno:
    - Sin key peg: Este color no se encuentra en el código
    - Key peg roja: Este color está en su posición adecuada
    - Key peg negra: Este color se encuenra en el código, pero está en una posición incorrecta.
